# Природа России — путешествие по самым красивым местам

**Командная работа Никиты и Арины**  
Конкурсная работа по вёрстке с нуля по методологии БЭМ (Nested)

## О проекте

Одностраничный адаптивный сайт-лендинг о самых живописных местах России: леса, горы, море и вдохновляющие истории путешественников.  
Полностью свёрстан вручную без использования фреймворков и сборщиков.

## Технологии и подходы

- HTML5 (семантическая вёрстка)
- CSS3 + полная адаптивность (320–1920+ px)
- Методология **БЭМ Nested** (строгая файловая структура)
- Normalize.css
- Подключение шрифтов через @font-face
- Pixel-perfect вёрстка по макету
- Кроссбраузерность: Chrome, Firefox, Яндекс.Браузер, Safari
- Валидный код (0 ошибок в валидаторе W3C)
